DATE_PARSEedit

Converts a string to a date, in the provided format. If no format is specified, the yyyy-MM-dd'T'HH:mm:ss.SSSZ format is used.

ROW date_string = "2022-05-06"
| EVAL date = DATE_PARSE(date_string, "yyyy-MM-dd")
